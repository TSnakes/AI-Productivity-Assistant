# AI Workplace Productivity Assistant

## Project Overview
This is an integrated, AI-powered workplace productivity platform built to automate everyday professional tasks. 

## Features Included
* **Smart Email Generator:** Instantly write professional emails with adjustable tones.
* **Meeting Notes Summarizer:** Condense long meeting texts into action items and deadlines.
* **AI Task Planner / Scheduler:** Generate daily/weekly schedules and prioritize tasks.
* **AI Chatbot Interface:** Interactive assistant to handle general workplace inquiries.

## Tools Used
* **Lovable.dev** for Frontend UI and AI logic generation.
* **GitHub** for version control.

## Responsible AI Disclaimer
This tool uses AI to assist productivity. Users should verify all outputs for accuracy before using them in a professional capacity.
